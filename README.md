# DigitRecognition
UI for predicting digits (0-9) model trained on MNIST DIGITS dataset.
Model I used is SVC(kernel='rbf',C=10).
Data has to be scaled with StandardScaler.
Accuracy achieved on MNIST DIGITS test_set is 0.9725
